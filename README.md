# One Stop - Project Management Company Website

A modern, professional, SEO-friendly website for One Stop, a project management company specializing in digital solutions.

## Features

- Modern, clean, and minimal design
- Fully responsive across all devices
- SEO-optimized with semantic HTML structure
- Fast-loading and lightweight
- Professional branding with custom color scheme
- Interactive contact form
- Smooth scrolling navigation

## Services

- Website Building
- Web App Development
- Business Process Management
- Branding

## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Modern ES6+ JavaScript

## Brand Colors

- Primary Black: #000000
- Primary Light: #FDFDFD
- Accent Orange: #F75F0C

## Getting Started

Install dependencies:

```bash
npm install
```

Build for production:

```bash
npm run build
```

## Project Structure

```
src/
├── components/
│   ├── Hero.tsx           # Hero section with main headline
│   ├── About.tsx          # Company introduction
│   ├── Services.tsx       # Services showcase
│   ├── WhyChooseUs.tsx    # Value propositions
│   ├── Contact.tsx        # Contact form and information
│   └── Footer.tsx         # Footer with social links
├── App.tsx                # Main application component
├── main.tsx               # Application entry point
└── index.css              # Global styles and Tailwind config
```
